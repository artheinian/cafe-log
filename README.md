# Café Log ☕

Café Log is a full-stack café drink tracker where users can sign up, log their favorite drinks, rate them, and manage their personal café diary.

## Built With

- React
- Vite
- Node.js
- Express
- SQLite
- JWT Authentication

## Features

- User sign up and login
- Invite-only registration
- JWT-based authentication
- Add, edit, and delete drinks
- Rate drinks from 1 to 5
- Organize drinks by category
- View personal drink history
- Public/private drink visibility

## Tech Overview

The frontend is built with React and Vite, while the backend uses Express and SQLite for API and data storage. Authentication is handled with JWT, and passwords are hashed with bcrypt.
